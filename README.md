# TypeScript 实时时钟网页

一个使用 TypeScript 构建的现代化实时时钟网页应用，具有数字和模拟时钟显示、主题切换、世界时间等功能。

## 功能特性

### 🕒 时钟显示
- **数字时钟**：实时显示时、分、秒，支持12/24小时制切换
- **模拟时钟**：带有平滑动画的模拟时钟，指针精确移动
- **日期信息**：显示完整日期、星期几、一年中的第几天

### 🎨 主题与交互
- **明暗主题**：一键切换亮色/暗色主题
- **响应式设计**：适配各种屏幕尺寸
- **平滑动画**：时钟指针和数字的平滑过渡效果
- **交互控制**：暂停/继续、时间格式切换

### 🌍 世界时间
- **多时区显示**：显示全球10个主要城市的时间
- **时区信息**：显示时区名称和UTC偏移量
- **自动更新**：所有时区时间实时同步更新

### 💻 技术特性
- **TypeScript**：强类型检查，更好的代码维护性
- **模块化架构**：清晰的代码分离和职责划分
- **现代化CSS**：CSS3动画、渐变、阴影效果
- **ES6+特性**：使用最新的JavaScript特性

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```
这将启动 TypeScript 编译监视模式和实时服务器。

### 构建项目
```bash
npm run build
```

### 启动项目
```bash
npm start
```

## 使用方法

1. **查看时间**：打开页面即可看到实时更新的时钟
2. **切换主题**：点击"切换主题"按钮在明暗模式间切换
3. **更改时间格式**：点击"12/24小时制"按钮切换时间显示格式
4. **暂停时钟**：点击"暂停"按钮停止时钟更新
5. **查看世界时间**：查看页面下方的世界主要城市时间

## 开发说明

### 代码架构
项目采用模块化设计，每个模块有明确的职责：
- `Clock`：处理时间计算和格式化
- `ThemeManager`：管理主题切换
- `TimezoneManager`：处理时区转换
- `UIUpdater`：更新DOM元素和动画

### 扩展功能
可以轻松扩展以下功能：
1. 添加更多时区
2. 实现闹钟功能
3. 添加计时器或倒计时
4. 集成天气信息
5. 添加多语言支持
